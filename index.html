Cross-Entropy vs. MSE — Binary Classification

An interactive explainer for students. Explore how the two loss functions behave.

Loss Function Definitions (Binary, y ∈ {0,1}, ŷ ∈ (0,1))

🔵 Cross-Entropy (Log Loss)

ℒCE = −[ y·log(ŷ) + (1−y)·log(1−ŷ) ]
If y = 1: −log(ŷ) — penalizes low confidence when true
If y = 0: −log(1−ŷ) — penalizes high confidence when false

Gradient w.r.t. ŷ:
∂ℒ/∂ŷ = (ŷ − y) / (ŷ(1−ŷ))

🟡 Mean Squared Error (MSE)

ℒMSE = (y − ŷ)²
If y = 1: (1 − ŷ)² — quadratic penalty
If y = 0: ŷ² — quadratic penalty

Gradient w.r.t. ŷ:
∂ℒ/∂ŷ = 2(ŷ − y)

🔵 Cross-EntropyGradient → ∞ as ŷ → 0 (confident wrong prediction). The model receives an increasingly large correction signal the more wrong it is.
🟡 MSEGradient = 2(ŷ − y), bounded between −2 and 2. The correction signal is always small — especially problematic when combined with a sigmoid activation.

Gradient w.r.t. the logit z, where ŷ = σ(z) — chain rule through the sigmoid

🔵 CE + Sigmoid

∂ℒCE/∂z = ∂ℒ/∂ŷ · σ′(z)
= [(ŷ−y) / ŷ(1−ŷ)] · ŷ(1−ŷ)
= ŷ − y

The σ′(z) term cancels exactly.
Gradient is just the prediction error — never vanishes.

🟡 MSE + Sigmoid

∂ℒMSE/∂z = ∂ℒ/∂ŷ · σ′(z)
= 2(ŷ−y) · σ′(z)
= 2(ŷ−y) · ŷ(1−ŷ)

σ′(z) stays in the expression.
Near z = ±∞, σ′(z) → 0 — gradient vanishes.

🔵 CE + Sigmoid — no vanishing gradient |∂ℒCE/∂z| = |ŷ−y|. When the model is maximally wrong (z → −∞ for y=1), ŷ→0 so the gradient → 1. Always a meaningful signal.
🟡 MSE + Sigmoid — vanishing gradient |∂ℒMSE/∂z| = 2|ŷ−y|·ŷ(1−ŷ). At the extremes σ′(z)→0, so the gradient collapses to 0 exactly when the prediction is most wrong.
